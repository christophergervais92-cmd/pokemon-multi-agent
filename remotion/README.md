# Remotion Video Project

This directory contains a Remotion project for creating programmatic videos.

## Getting Started

1. Install dependencies:
```bash
cd remotion
npm install
```

2. Start the Remotion Studio (development mode):
```bash
npm run dev
```

This will open Remotion Studio in your browser where you can preview and edit your videos.

## Available Scripts

- `npm run dev` - Start Remotion Studio for development
- `npm run build` - Bundle the Remotion project
- `npm run render` - Render a video (requires additional parameters)
- `npm run upgrade` - Upgrade Remotion to the latest version

## Creating Videos

Edit the compositions in `src/Root.tsx` and create new components in `src/` to build your videos.
